# Generated Concept Prompt

Mode: built-in image generation.

The generated concept board is intentionally looser than the deterministic HTML
prototype. It explores mood, typography, and palette. It is not intended to
preserve the exact brand mark or serve as an implementation specification.

```text
Use case: ui-mockup
Asset type: design audit concept board for an internal cafe operations dashboard
Primary request: Create an alternate visual direction for the Blue Poppy Ops
dashboard, using the two supplied dashboard images as information-architecture
references. This is a design exploration, not a production screenshot.
Input images: Image 1 is the proposed desktop dashboard reference; Image 2 is
the proposed mobile dashboard reference.
Scene/backdrop: a clean warm-light-grey product design presentation board.
Subject: one large desktop dashboard mockup and one mobile dashboard mockup,
showing live daily takings, hourly sales bars, weekly/monthly KPI cards, and a
concise morning brief.
Style/medium: high-fidelity modern product UI, editorial and operational,
restrained cafe warmth rather than futuristic SaaS.
Composition/framing: desktop mockup dominant on the left, mobile mockup
overlapping slightly on the right, both shown straight-on without photorealistic
devices. Add small color and typography swatches along the bottom.
Color palette: matte charcoal and ink-black surfaces, warm cream typography,
muted cornflower blue data accents, sage green positive indicators, a very
small amount of warm tan.
Text (verbatim): "Today at a glance", "$3,842", "Morning brief", "$18,460".
Keep all other microcopy minimal or abstract rather than generating gibberish.
Constraints: strong information hierarchy; the live takings number is the
unmistakable focal point; clear status/freshness cue; accessible contrast; calm
navigation; credible bar charts and sparklines; desktop and mobile should feel
like the same system.
Avoid: neon, cyberpunk, glassmorphism, glossy 3D devices, stock photography,
illustrations, excessive gradients, excessive rounded pills, clutter, tiny
illegible text, fake paragraphs, watermarks.
```
